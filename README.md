# sockchain
A configurable command line / terminal cryptocurrency ticker written in node

Looks something like...

```
┌────────┬─────────┬───────┬───────┐
│ Symbol │  Price  │  1h   │  24h  │
├────────┼─────────┼───────┼───────┤
│  BTC   │ 3172.93 │ -1.07 │ 0.77  │
├────────┼─────────┼───────┼───────┤
│  ETH   │  254.0  │ -0.44 │ 9.04  │
├────────┼─────────┼───────┼───────┤
│  LTC   │ 45.4451 │ -0.68 │ -0.82 │
└────────┴─────────┴───────┴───────┘
```

# Getting Started

1. First install Node.js
2.

```
npm install
npm start
```
